# Future Improvements

## Centralized Logging

Aggregate logs from pfSense, Proxmox, and important virtual machines into a central logging platform for improved visibility and correlation.

## SIEM Integration

Add a SIEM or security monitoring platform to improve alerting, log retention, and incident review workflows.

## Reverse Proxy and TLS

Deploy a reverse proxy with proper TLS configuration for internally hosted services that need cleaner and more secure access patterns.

## Vulnerability Scanning

Introduce scheduled vulnerability scans against lab assets to identify weak configurations and outdated software.

## Endpoint Monitoring

Add endpoint-level security telemetry for critical systems and high-value virtual machines.

## Configuration Backup and Change Control

Document and version control important firewall, switch, and hypervisor configuration changes to improve recovery and auditability.

## Expanded Segmentation

Add additional zones for:

- guest access
- backup network
- security tooling
- isolated malware analysis or test workloads
